⚠️ WARNING: Complete reset of the password manager

Associated risks:

  • - This operation will reset all settings related to the generation of your passwords.
  • - Any change, even a minor one (a single character in your username or master password), will make it impossible to regenerate existing passwords for already saved sites.
  • - No going back after confirmation.

Possible consequences:

  • - Loss of access to your online accounts if you do not know the current passwords.
    • - Need to manually reset each password on each affected site.

Recommendations before continuing:

  1. Back up the hidden id.json file to an external device (USB drive, hard drive, secure cloud).
  2. Make sure you have a copy of all your important passwords.
  3. If you are not a technician or are unsure of the consequences, seek professional help.
